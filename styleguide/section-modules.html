Wrap #Modules.wrap

Include wrap function inside container class to hold content in maximize width with space aside corner. Space can be created in asix x (padding left and padding-right).
Default class for this function named .wrap and is located in style/_tools.scss file. Prepared modifications are navbar and footer.

Source code located in file playground/index.scss .

Function

add-wrap($max-width, $padding-x);

Arguments

Variable Type Default value
$max-width numberPixel
$padding-x numberPixel? 0

Usage

.wrap {
 @include add-wrap(1024px, 20px);
}

Output

Pug

.wrap

HTML

<div class="wrap"></div>
